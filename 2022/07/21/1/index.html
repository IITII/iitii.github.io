下载工具以及镜像

  • platform-tools(adb): https://developer.android.com/studio/releases/platform-tools
    • mac: brew install --cask android-platform-tools
    • 如果不能在 cli 里面访问到 adb 之类的命令，请自行添加环境变量
  • MIUI解锁工具: miui.com/unlock/
  • TWRP: https://twrp.me/xiaomi/xiaomipocof3.html
    • 美国镜像站：https://dl.twrp.me/alioth/
    • 欧洲镜像站：https://eu.dl.twrp.me/alioth/
  • Magisk:
    • https://github.com/topjohnwu/Magisk/releases
  • 欧洲版MIUI包(eu)：https://androidfilehost.com/?w=files&flid=331502
    • 打开搜索 K40, POCO_F3
    • https://androidfilehost.com/?fid=15664248565197180206
  • Magisk模块
    • SafetyNet(网飞检测)：https://github.com/kdrag0n/safetynet-fix
    • Zygisk：无需额外下载

国际版没有通话录音、来电拦截、短信拦截之类的功能，没法用。EU版用的是小米自带的拨号和联系人，支持通话录音

刷入

刷机10分钟，恢复两三天

  1. 解压 fastboot 刷机包
  2. 手机重启到 fastboot：adb reboot fastboot
  3. 因为这次是从国际版刷机包EU版，所以备份数据+双清稳一点
    1. 不保留数据：windows_fastboot_first_install_with_data_format.bat
    2. 保留数据：windows_fastboot_update_rom.bat
  4. 等待命令跑完，刷机完成
  5. 初始化+备份恢复

备份与恢复

eu 版用的 MIUI 的拨号和联系人，可以正常恢复。再加 Ti备份之类的就更靠谱了

  1. 系统设置 里面直接搜索 备份，就能找到备份与恢复 app
  2. 将之前备份的文件拷贝回手机
  3. 恢复

下面操作和之前是一样的

Magisk

https://topjohnwu.github.io/Magisk/install.html

  1. adb install Magisk App 或者拷贝后手动安装
  2. 打开卡刷包，找 boot.img，拷贝到手机
  3. Magisk App点击Magisk，手动选择刚刚拷贝的 boot.img，运行完成后应该会生成一个 patch.img 文件
  4. 拷贝patch.img 文件到电脑
  5. 进 bootloader
  6. 先尝试使用 patch.img 文件是否可以正常启动
  7. 可以正常启动后将 patch.img 文件刷入 boot 分区

安装 Zygisk

  1. Magisk App点击设置
  2. 勾选在 Zygisk 里面运行 Magisk
  3. 自动下载，安装
  4. 重启

SafetyNet

  1. 下载 zip 包到手机
  2. Magisk里面选择安装，重启
  3. 基本检测通过大概率没啥问题

Magisk/Root 隐藏

  1. Magisk App点击设置
  2. 不勾选”遵守排除列表“
  3. 点击 ”配置排除列表“
  4. 搜索需要隐藏的软件（大部分时候是银行App和金融App

一个App有很多项进程，都需要勾上，点击即可展开，再点击勾选即可全选。

MIUI EU 欧洲版 本地化 Magisk 模块 - Koizumi’s Blog

  • magisk 选择安装这个zip
  • 使用音量键进行选择选项
  • 刷入重启
